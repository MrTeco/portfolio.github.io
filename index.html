Anemia

A 2D topdown multiplayer action game in Unity

I started this project in late 2019 when I got really interested in Space Station 13, I took a lot of inspiration from that game and decided to create something unique that I can play with others. I made this using Unity and the Mirror Networking asset which allowed me to semi-easily implement multiplayer. I always wanted a very "grimdark" atmosphere type of game with horror and survival elements, so I took inspiraton from things like the Navajo Skinwalker folklore and other games with a similar feel. I hope that I can one day get a small active community of players that I can play with where the community gets to shape the world in a way.

I spent a LOT of time on this project, and I want to continue advancing on it until it becomes a fully fledged fun and enjoyable game.

The Unity project is getting pretty large, and I've considered switching to my own game engine before. So it does sometimes get a little too bloated but new coding practices I learn and things such as ECS - Entity Component System structure help me out.

Discord Group

Cadmium and "cadmiumbot"

A server mod for Barotrauma and a Discord bot in C#

This is a plugin for a game called Barotrauma, I made it when I realized that the game is made in C#, open-source and mod friendly. It offers players a new experience, because it features things like added custom gamemodes, quality of life features like chat commands and currency, and sometimes completely changes how the game works.

And cadmiumbot is the Discord Bot that I made, it reads the clients points on a .txt file on my pc, and displays a leaderboard in a Discord channel. I made it by using "discord.net" which are like C# bindings for making bots and using the bot api.
